# Communication-Networks
This repository contains laboratory exercises from the course [Communication Networks](https://www.fer.unizg.hr/en/course/comnet_b) at FER.

## About
Modern computing is inseparable from networking. This course provides an introduction to communication networks, covering network architectures, classification, and topology. Core topics include layered reference models (OSI and TCP/IP), the Internet Protocol (IP), transport layer protocols (TCP and UDP), application layer protocols and services (DNS, HTTP, email), fundamentals of network security, and the evolution of internet technologies from local to global scale.

## Learning Outcomes
- define concept, architecture, and organisation of communication networks
- explain how communication networks operate and their functionality
- apply knowledge about communication networks and protocols
- analyze protocol functions and services to select suitable solutions
- analyze organization of public and private networks based on the IP protocol
- define security threats and available solutions in the Internet
- design network models including local area networks, Internet subnetworks, and Internet access
- evaluate communication solutions based on the TCP/IP protocol stack
